# She Can Foundation Website

A responsive and modern NGO website created as part of a Web Development Internship Task for She Can Foundation.

## 🌟 Overview

This website is designed to spread awareness about the mission of She Can Foundation and encourage people to volunteer and contribute to social impact initiatives.

The project focuses on clean design, responsiveness, accessibility, and user engagement.

## 🚀 Features

- Responsive Navigation Bar
- Hero Section with Call-to-Action
- About Us Section
- Impact Statistics Section
- Mission & Vision Section
- Volunteer Testimonials
- Volunteer Registration Form
- Contact Information Section
- Smooth Scrolling Navigation
- Scroll Reveal Animations
- Mobile-Friendly Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- ScrollReveal.js
- Google Fonts (Poppins)

## 📸 Website Sections

### Home
Introduces the organization and encourages visitors to become volunteers.

### About Us
Provides information about the foundation's purpose and goals.

### Impact
Highlights the positive impact created through volunteers and programs.

### Mission & Vision
Explains the foundation's mission and long-term vision.

### Testimonials
Displays feedback from volunteers.

### Volunteer Form
Allows interested individuals to express interest in volunteering.

### Contact
Provides contact information for further communication.

## 📱 Responsive Design

The website is fully responsive and works across:

- Desktop
- Laptop
- Tablet
- Mobile Devices

## 🎯 Objective

The objective of this project is to create a simple yet visually appealing website that represents the values and mission of She Can Foundation while encouraging community participation.
